## 🌟 Overview

Welcome to my personal showcase. This digital portfolio is the result of a quest to merge technical precision with creative expression. It serves as a living document of my journey as a developer and designer, providing a deep dive into my work, my technical approach, and my aesthetic vision.

## 🎨 Design Philosophy

- **Minimalism with Purpose**: Every element on the page exists to serve a function or highlight content. Minimal clutter ensures the work takes center stage.
- **Micro-Interactions**: Subtle animations and transitions create a sense of life and responsiveness, making the digital experience feel more tactile and premium.
- **Typography & Grid**: A strong focus on typographic hierarchy and a rigid grid system provides structure and readability across all devices.
- **High Contrast**: Using a deep dark palette with luminous highlights for a refined, modern "developer" aesthetic.

## ✨ Features

- **🎨 Modern Design**: Sleek, minimalist UI with smooth animations and glassmorphism.
- **🌍 Localization**: Full support for English and Italian using `i18next`.
- **📱 Responsive**: Perfectly optimized for all screen sizes, from mobile to ultra-wide.
- **⚡ Performance**: Lightning-fast load times thanks to Vite and optimized assets.
- **🌗 Dark Mode**: Thoughtfully crafted dark theme for a premium developer experience.

## 🏗️ Project Structure

```text
src/
├── assets/         # Optimized images and brand assets
├── components/     # Reusable UI building blocks (shadcn/ui + custom)
├── i18n/           # Internationalization configuration and resources
├── lib/            # Utility functions and shared library configurations
├── pages/          # Full page views and routing logic
└── App.tsx         # Main application entry and router setup
```

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **UI Components**: Shadcn/UI, Radix UI, Lucide Icons
- **State/Data**: TanStack Query (React Query)
- **Internationalization**: i18next, react-i18next
- **Build Tool**: Vite

## 🌍 Localization

The project uses `i18next` for managing translations. Currently supported languages:
- 🇮🇹 **Italian (it)**
- 🇬🇧 **English (en)**

Translations can be found in `src/i18n.ts`.
